# PĒÑDØ CBT

PĒÑDØ CBT is a Computer-Based Test (CBT) platform developed for students to practice examinations online.

## Features
- Student Registration
- Student Login
- CBT Examination
- Countdown Timer
- Automatic Scoring
- Result Page
- Mobile Friendly
- Anti-Cheating Features

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages
